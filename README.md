# To Do APP

## Описание проекта

В приложении реализовано добавление задач, их выполнение и удаление

При попытке добавить задачу с пустым полем ввода, кнопка добавления неактивна

Стек:
- React Typescript
- Redux Toolkit
- Material UI (1 иконка крестика)

## Начало работы

Используйте:

### `npm install`
### `npm run start`
